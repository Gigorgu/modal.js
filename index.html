Modal.JS

A free, lightweight jQuery Modal plugin. It is only ~1KB gzipped. Configurable using direct options and HTML5 data-* attributes.

View on Github Download
Version 1.0
Get Started
Danger , Do not use bootstrap with this plugin right now!
Step by step create the first modal instance.
jquery-3 jquery - is default plugin for js , if you not include jquery . Modal plugin may be not work 
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
Css You can create your stylesheet , but if you use default styles , all be okay 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Gigorgu/app.modal.js/modal.min.css">
Javascript 
<script src="https://cdn.jsdelivr.net/gh/Gigorgu/app.modal.js/modal.min.js"></script>
Init 
$('.callmodal').modal()
Options The default modules for use , and you can know
Value Handle
data-container="$" $ - your value , data-container sets to container of modal class or id
data-modal="$" $ - your value , data-modal sets to modal of class or id
close="" Close can be used in modal , he search modal mains and closes
How it works You can see the sample 

<button data-container=".container_modal" data-modal="#target" class="btnmodal">Modal Button</button>
<script>
    $('.btnmodal').modal()
</script>
<!-- AFTER THIS -->
<!-- MODAL -->
<div class="container-modal">
    <div id="target">
    <!-- CONTENT -->
        <!-- header text and close button -->
        <div class="__header"></div>

        <!-- text -->
        <div class="__text"></div>

        <!-- bottom group - may be use for buttons -->
        <div class="__text"></div>
    </div>
</div>
